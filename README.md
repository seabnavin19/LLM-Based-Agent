## About this project
It is a demo app of AI travel agent that we user hotel and event API as a tool to demo the capability of AI agent 

## How to start
1. clone the project
2. create venv 
   ```
   python -m venv venv
   ```
3. install library
   ```
   pip install -r requirements.txt
   ```
4. add API key in .env
GEMINI_API_KEY = ""
SERP_API_KEY = "3"

5. start project
   ```
   streamlit run app.py
   ```
